About Report Formats

At any point in time, a report can have only one report format. Pentaho Analyzer offers the following report formats:

  • Table
  • Standard Charts:
    • Column
    • Stacked Column
    • 100% Stacked Column
    • Column-Line Combo
    • Bar
    • Stacked Bar
    • 100% Stacked Bar
    • Line
    • Pie
    • Area
    • Scatter
    • Heat Grid
    • Geo Map

Switching Report Formats

Select one of the following methods to switch the report format:

  • Click the chart icon, located in the upper right corner of the report, and select a chart format from the menu.
  • Click the Show as Chart or Show as Table button, located in the upper right corner of the report, to toggle between the table format and the default chart format. Note: Depending on the current format of the report, only one of the buttons (Show as Table OR Show as Chart) is available at a time.

Hiding Measure Fields When Viewing Charts

When you present your data using a chart (e.g. line chart), you often want to display only the most important measure field(s) (e.g. Profit) to avoid cluttering the chart. However, you might want to be able switch back to the table format to get the exact numbers and view some additional measure fields (e.g. Revenue, Cost). Analyzer supports this by allowing you to hide measure fields in the chart formats.

Select one of the following methods to hide a measure field in the chart format:

  • If you currently are viewing the report in table format, click the field you want to hide on the report and select Hide from Chart from the menu.
  • If you currently are viewing the report in a chart format, click the field button in the Field Layout panel and select Show on Chart from the menu.

In the Field Layout panel, fields that currently are hidden are gray in color (as opposed to the blue color of visible fields).

To show a field again, click the field you want to make visible and select Show on Chart from the menu.
